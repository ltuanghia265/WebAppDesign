Câu 1. Kiến trúc client-server gồm 4 thành phần sau:

A. Server, Services, Môi trường truyền, Giao thức truyền thông tin
B. Client, Server, Môi trường truyền, Giao thức truyền thông tin
C. Client, Server, Môi trường truyền, Thiết bị lưu trữ
D. Client, Server, Trình duyệt, Giao thức truyền thông tin

Câu trả đúng là:
B. Client, Server, Môi trường truyền, Giao thức truyền thông tin.
Thành phần Client là máy khách, được sử dụng để gửi yêu cầu đến máy chủ. Thành phần Server là máy chủ, được sử dụng để cung cấp dịch vụ cho máy khách. Thành phần Môi trường truyền là môi trường mà thông tin được truyền qua lại giữa máy khách và máy chủ. Thành phần Giao thức truyền thông tin là các quy tắc và định dạng được sử dụng để giao tiếp giữa máy khách và máy chủ 1.

Câu 2. Trong kiến trúc client-server, client có thể là trình duyệt, ứng dụng viết bằng python/javascript hoặc bất kỳ ứng dụng nào mà có phát sinh ra ______.

A. Kết quả
B. HTTP request
C. Giao thức
D. HTML

Câu trả đúng là:
Trong kiến trúc client-server, client có thể là trình duyệt, ứng dụng viết bằng python/javascript hoặc bất kỳ ứng dụng nào mà có phát sinh ra HTTP request .
Trong kiến trúc client-server, client là thành phần gửi yêu cầu đến máy chủ. Để gửi yêu cầu này, client sử dụng giao thức HTTP (HyperText Transfer Protocol) để tương tác với máy chủ.

Câu 3. Một số phần mềm dùng để tạo ra HTTP Web Server gồm:

A. IIS, Apache, Tomcat, NodeJS, Unix, GWS, Tornado
B. IIS, Apache, Tomcat, DNS, Nginx, GWS, Tornado
C. IIS, Apache, Tomcat, NodeJS, Nginx, GWS, HTTP response
D. IIS, Apache, Tomcat, NodeJS, Nginx, GWS, Tornado

Câu trả đúng là:
Một số phần mềm dùng để tạo ra HTTP Web Server bao gồm:
A. IIS, Apache, Tomcat, NodeJS, Unix, GWS, Tornado.
IIS (Internet Information Services) là một máy chủ web được phát triển bởi Microsoft. Apache là một máy chủ web mã nguồn mở được sử dụng rộng rãi trên nhiều hệ điều hành. Tomcat là một máy chủ ứng dụng web mã nguồn mở được phát triển bởi Apache Software Foundation. NodeJS là một nền tảng phát triển được xây dựng trên Chrome’s JavaScript runtime để xây dựng các ứng dụng mạng có khả năng xử lý đa luồng. Unix là hệ điều hành đa nền tảng được sử dụng rộng rãi trong các máy chủ web. GWS (Google Web Server) là một máy chủ web được phát triển bởi Google. Tornado là một máy chủ web mã nguồn mở được sử dụng để xây dựng các ứng dụng web thời gian thực

Câu hỏi 4. Client–server model is a distributed application structure that partitions tasks or workloads between the providers of a resource or service, called _______, and service requesters, called clients.

A. browsers
B. web services
C. servers
D. centers

Câu trả đúng là:
The Client-server model is a distributed application structure that partitions tasks or workloads between the providers of a resource or service, called servers, and service requesters, called clients. Therefore, the answer to your question is
C. servers.
