Câu 1. DNS là viết tắt của:
A. Domain Name Services
B. Domain Name System
C. Domain Network Services
D. Document Name System

DNS là viết tắt của Domain Name System. Vì vậy, câu trả lời chính xác nhất là:
B. Domain Name System

Câu 2. The DNS is a hierarchical and _________ naming system for computers, services, or other resources connected to the Internet or a private network.
A. centralized
B. domain
C. decentralized
D. digital

The DNS is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network. So, the correct answer is:
C. decentralized

Câu 3. Hệ thống DNS gồm 3 thành phần là:
A. Name server, DNS namespace và Resolver
B. DNS system, Resolver và Name server
C. Root server, DNS namespace, và Name server
D. DNS namespace, Domain Name và Resolver

Hệ thống DNS gồm 3 thành phần chính là: Name server, DNS namespace, và Resolver. Vì vậy, câu trả lời chính xác nhất là:
A. Name server, DNS namespace và Resolver

Câu 4. Dấu chấm ở cuối tên miền “http://www.vidu.com.”, mang ý nghĩa gì?
A. Không mang ý nghĩa gì, do viết thừa
B. Tên miền gốc (root domain)
C. Tên miền mức cao (top level domain)
D. Thư mục gốc của website

Dấu chấm ở cuối tên miền “http://www.vidu.com.” mang ý nghĩa là Tên miền gốc (root domain). Vì vậy, câu trả lời chính xác nhất là:
B. Tên miền gốc (root domain)

Câu 5. Tổ chức quản lý và cung cấp tên miền quốc tế và Việt Nam là:
A. W3C và VNPT
B. eBay và Mắt bão
C. ICANN và VNPT
D. ICANN và VNNIC

Tổ chức quản lý và cung cấp tên miền quốc tế là ICANN và tên miền Việt Nam là VNNIC. Vì vậy, câu trả lời chính xác nhất là:
D. ICANN và VNNIC

Câu 6. Trong hệ điều hành Windows, tập tin hosts chứa cái gì?
A. Danh sách các máy chủ
B. Địa chỉ các máy host
C. Các bản ghi IP-Domain name
D. Địa chỉ các DNS server

Trong hệ điều hành Windows, tập tin hosts chứa các bản ghi IP-Domain name. Vì vậy, câu trả lời chính xác nhất là:
C. Các bản ghi IP-Domain name
