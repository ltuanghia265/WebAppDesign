Câu 1. Câu nào phát biểu đúng nhất về client-side và server-side?
A. Client-side gồm các hoạt động tại trình duyệt, server-side gồm các hoạt động tại web server
B. Client-side gồm các hoạt động tại web server, server-side gồm các hoạt động tại trình duyệt
C. Client-side nhận yêu cầu từ web server, server-side gửi yêu cầu tới trình duyệt
D. Client-side gắn với máy chủ, server-side gắn với trình duyệt

Đáp án đúng là:
A. Client-side gồm các hoạt động tại trình duyệt, server-side gồm các hoạt động tại web server
Client-side và server-side là hai khía cạnh quan trọng của phát triển web. Client-side (hay còn gọi là phía người dùng) liên quan đến tất cả những gì diễn ra trên trình duyệt của người dùng, trong khi server-side (hay còn gọi là phía máy chủ) liên quan đến những gì diễn ra trên máy chủ web.

Câu 2. Câu nào phát biểu đúng nhất về front-end và back-end?
A. Front-end gồm các xử lý tại máy server, back-end gồm giao diện và các xử lý tại giao diện của ứng dụng
B. Để làm phần front-end cần sử dụng một trong các ngôn ngữ C#, PHP, JavaScript, Ruby, Python
C. Front-end gồm giao diện và các xử lý tại giao diện của ứng dụng, back-end gồm các xử lý tại máy server
D. Để làm phần back-end có thể sử dụng các ngôn ngữ HTML, CSS, JavaScript

Đáp án đúng là:
C. Front-end gồm giao diện và các xử lý tại giao diện của ứng dụng, back-end gồm các xử lý tại máy server
Front-end (hay còn gọi là phía người dùng) liên quan đến việc tạo ra và thiết kế giao diện người dùng, cung cấp trải nghiệm người dùng mượt mà trên trình duyệt. Ngôn ngữ chính được sử dụng cho front-end development bao gồm HTML, CSS và JavaScript.
Back-end (hay còn gọi là phía máy chủ) liên quan đến việc xử lý logic nền tảng của ứng dụng, thao tác với cơ sở dữ liệu, xử lý yêu cầu từ front-end và gửi phản hồi phù hợp. Các ngôn ngữ lập trình phổ biến cho back-end development bao gồm Java, Python, Ruby, Node.js (JavaScript), PHP và .NET (C#).

Câu 3. A web server is computer software and underlying hardware that accepts requests via______, the network protocol created to distribute web pages, or its secure variant HTTPS.
A. DNS
B. JavaScript
C. HTML
D. HTTP

Đáp án đúng là:
D. HTTP
Một máy chủ web là phần mềm máy tính và phần cứng bên dưới hỗ trợ các yêu cầu thông qua HTTP, giao thức mạng được tạo ra để phân phối các trang web, hoặc phiên bản an toàn của nó là HTTPS.

Câu 4. Web root is the topmost _______in which the files making up a web site are located on a web server.
A. file
B. directory
C. server
D. protocol

Đáp án đúng là:
B. directory
Web root là thư mục cao nhất trong đó các tệp tạo nên một trang web được đặt trên máy chủ web

A. 127.0.0.0
B. 127.0.0.1
C. 8.8.8.8
D. 192.168.0.1

Địa chỉ IP mặc định của web server khi cài trên máy cục bộ là 127.0.0.1. Vì vậy, câu trả lời chính xác nhất là:

Câu 6. Tên miền mặc định của web server, khi cài trên máy cục bộ là?
A. webserver
B. root
C. webroot
D. localhost

Tên miền mặc định của web server khi cài trên máy cục bộ là localhost. Vì vậy, câu trả lời chính xác nhất là:
D. localhost
