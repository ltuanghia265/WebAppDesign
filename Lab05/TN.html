Câu 3. In computer science, an _______ is a computer program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program.

A. compiler
B. interpreter
C. assembler
D. coder

Câu trả lời đúng là:
In computer science, an interpreter is a computer program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program . Therefore, the correct answer is B. interpreter.

Câu 4. In computing, a ________ is a computer program that translates computer code written in one programming language (the source language) into another language (the target language).

A. translator
B. interpreter
C. compiler
D. assembler

Câu trả lời đúng là:
In computing, a compiler is a computer program that translates computer code written in one programming language (the source language) into another language (the target language). Therefore, the correct answer is C. compiler.

Câu 5. Để làm ra một trang web bán hàng, cách làm nào là nhanh nhất trong những cách sau?

A. Lập trình bằng hợp ngữ (assembly)
B. Sử dụng framework
C. Sử dụng CMS
D. Lập trình bằng ngôn ngữ cấp cao (thuần)

Câu trả lời đúng là:
Để làm ra một trang web bán hàng, cách nhanh nhất trong những cách sau là sử dụng CMS. CMS là viết tắt của Content Management System, là một hệ thống quản lý nội dung giúp bạn xây dựng và quản lý trang web một cách dễ dàng. CMS giúp bạn tạo ra một trang web bán hàng mà không cần phải viết mã từ đầu. Một số CMS phổ biến bao gồm WordPress, Shopify, Squarespace, Square Online và Wix
